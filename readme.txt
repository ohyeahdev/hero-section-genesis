== Description ==

With WordPress Customizer you can add a new hero section in your homepage.

Settings:

   * Set a background section image.
   * Adjust the image height.
   * Set the title of hero section, font size and text color.
   * Set the subtitle of hero section, font size and text color.

<a href="https://www.valhallawp.com/plugin/hero-section-genesis" target="_blank">More info</a>

Note: This plugin requires Genesis Framework

== Installation ==

1. Visit your admin section and click Plugins -> Add New.  Then search for "Hero Section Genesis”.

2. After activate it, from Appearance -> Customize -> Hero Section Genesis -> Hero Section 1 you can adjust the settings.
